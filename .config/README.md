# Notification Manager {dunst}
## Install dunst && Create folder for dunst
```bash
sudo pacman -S dunst
mkdir ~/.config/dunst/
```
## Copying Default Config to .config
```bash
cp /usr/share/dunst/dunstrc ~/.config/dunst/dunstrc
```

# Fish Shell
## install fish
```bash 
sudo pacman -S fish
```
## Changing Shell
```bash
fish
chsh /usr/bin/fish
```
## Configure Fish Shell
```bash
fish_config
```
